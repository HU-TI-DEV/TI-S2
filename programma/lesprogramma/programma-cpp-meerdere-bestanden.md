## Voorbereiding
- Het is verstandig om een **[nieuwe workspace](https://code.visualstudio.com/docs/editing/workspaces/workspaces)** aan te maken binnen VS code. 
- Maak een backup van je huidige VS code propertiefiles (*.json).
- Lees de informatie over [meerdere bronbestanden](../../software/c++/meerdere-bronbestanden/README.md).


### Sneak peak
We gebruiken meerdere files voor de teken-opdracht met SFML. 
Kijk alvast naar de [installatie van SFML](../../software/inrichten-ontwikkelomgeving/sfml_installatie.md). 
Zoals al eerder vermeld: het is handig om hier *een aparte workspace binnen VS code* voor aan te maken.

## Tijdens de les
1. De docent licht het het gebruik van meerdere bestanden binnen C++ toe.
2. Daarna kijken we naar de configuratie van de VS code ontwikkelomgeving.
3. We compileren en runnen meerdere files.
4. Bij tijd over: we kijken *kort* naar [godbolt.org](https://godbolt.org/)

## Help!
Als niets meer werkt, gebruik dan *tijdelijk* een online omgeving: 
- [cpp.sh](https://cpp.sh/) of 
- [online compiler](https://www.programiz.com/cpp-programming/online-compiler/).

3. Vraag *feedback* op je code tijdens de les.
