# Inrichten ontwikkelomgeving

In deze les gaan we een ontwikkelomgeving opzetten. 

## Voorbereiding

- Neem een laptop mee naar de les

## Tijdens de les

Gaan we een ontwikkelomgeving opzetten. Alles wat je nodig hebt en hoe je dit moet installeren kan je vinden onder bij de uitleg [Inrichten ontwikkelomgeving](../../software/inrichten-ontwikkelomgeving/README.md).

## Na de les

Heb je een compiler zodat je C++ (desktop)programma's kunt compileren. Heb je een werkende Intergrated Development Environment (IDE).

[Je kunt een "Helo World" compileren](../../software/c++/basisconcepten/README.md).

 Kan je een eenvoudig programma maken en flashen op een Arduino Uno met de Arduino IDE.

### Canvas
- Lever het resultaat in op de Canvas entry voor deze opdracht "Werkende Ontwikkelomgeving".
