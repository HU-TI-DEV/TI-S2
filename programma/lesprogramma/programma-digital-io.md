# Samenvatting ["Digital-I/O"](../../hardware-interfacing/communicatie/analoog-en-digitaal/Digital-IO/README.md)

## Hardware

* 1 x Arduino UNO setje met
  * 1 x Arduino UNO
  * 1 x USB-kabel USB-A tot USB-B (krijg je met het Arduino UNO)
* 1 x Adapter USB-C tot USB-A (als jouw laptop geen USB-A-aansluitingen heeft)
* 1 x Breadboard
* 1 x Button
* 1 x LED (bijvoorbeeld LED 5mm rood)
* 1 x weerstand 1 kOhm
* 6 x draadjes (male-male)

## Na deze les

* heb je een werkend Arduino IDE op jouw laptop
* kan je met de Arduino IDE een Arduino Uno met een programma flashen
* kan je in jouw team op een breadboard een eenvoudige schakeling opbouwen om met een [button (knop) een LED aan te sturen](../../hardware-interfacing/communicatie/analoog-en-digitaal/Digital-IO/README.md)
* weet je dat je op een microcontroller (Arduino UNO) voor het inlezen van digitale signalen een ingebouwde pull-up weerstand kan activeren
* weet je dat je een LED altijd met een weerstand gebruikt
